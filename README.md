# machine_learning_basics
Student Career Prediction is a machine learning–ready project that analyzes student skill levels in Python, SQL, and Java to predict their future career path. The dataset has been cleaned, explored, and structured for supervised multi-class classification. The project is currently prepared for model training and evaluation.

**📌 Project Overview**
.This project aims to build a Machine Learning based Career Prediction System that predicts a student's future career path based on their technical skill levels.
.Currently, the project has completed:
.Data preprocessing
.Exploratory Data Analysis (EDA)
.Feature selection
.Problem formulation as a Machine Learning task
.The project is now structured and ready for model training.

**🎯 Problem Statement**
To predict a student's Future Career using their programming skill levels.

📥 Input Features:
Python skill level
SQL skill level
Java skill level

🎯 Target Variable:
Future Career (categorical)

**🧠 Machine Learning Formulation**

After data exploration, the problem has been defined as:
Supervised Multi-Class Classification Problem
Because:
Input → Numerical skill ratings
Output → Categorical career labels
